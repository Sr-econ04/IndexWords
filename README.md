# はば単 Index Words

兵庫県の中学生向け英単語帳「はば単（第２版）」を使った探索型英単語ゲームです。


## ゲームの概要

「書ける」よりも「知っている・読める」を重視した学習ゲームです。

正解の単語をキーボードで入力して探索します。回答するたびに探索範囲がせばまり、バーで正解の位置が確認できます。少ない手数で当てるほど高スコアになります。


## あそびかた

1. トップページで「単語データを読み込む」ボタンを押す
2. 品詞（全単語・名詞・動詞・形容詞・副詞）を選ぶ
3. キーボードで単語を入力してENTERで送信する
4. 探索範囲のバーと残り候補数を見ながら絞り込む
5. 正解するとスコアとランクが表示される


## スコアとランク

理論値は ceil(log2(候補数)) で計算されます。たとえば全単語モードでは約12手が理論値です。

- S　理論値より2手以上少ない
- A　理論値の前後1手以内
- B　理論値より2〜3手多い
- C　理論値より4〜5手多い
- D　理論値より6手以上多い


## 単語データについて

単語データはゲーム起動時にボタンを押すことで、兵庫県教育委員会の公式サイトから直接取得します。アプリ内に単語データは含まれていません。

取得元　https://www2.hyogo-c.ed.jp/hpe/uploads/sites/10/2023/03/R3ver2_Habatanforstudents.xls

単語データの著作権は兵庫県教育委員会に帰属します。
